# TopoMaskV3.github.io

Initial GitHub Pages scaffold for the TopoMaskV3 paper project.

## Included in this first version

- Single-page project layout for abstract, method, benchmark, and resources.
- Figure assets converted from `../TopoMaskV3_revised/main_figures/*.pdf` into `assets/images/`.
- Links to benchmark resources in `OpenLane-V2` branch `different_splits`.
- Paper/arXiv buttons intentionally disabled until public release.

## Local preview

Open `index.html` directly in a browser or use any static server from this repo root.
